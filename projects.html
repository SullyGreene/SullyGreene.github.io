📟 MyPager — CLI Toolkit

MyPager is a Python command-line tool designed to simplify script management. Whether you're running automation or installing quick helpers, MyPager’s got you! 💻🚀

  • 🛠️ Easy script execution with run/install commands
  • 💻 Interactive menu mode for browsing and managing tools
  • 🎨 Easily extendable to your workflow
python MyPager.py run my_script

Clone & start:

git clone https://github.com/SullyGreene/MyPager.git
cd MyPager
python MyPager.py
